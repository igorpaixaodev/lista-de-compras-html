# 🛒 Minha Lista de Compras

> Um projeto desenvolvido sob demanda de um **stakeholder muito exigente** (minha esposa 👰‍♀️)

## 📖 Sobre o Projeto

Este é um aplicativo web simples e elegante para gerenciar listas de compras. Nasceu de uma necessidade pessoal quando minha esposa começou a pedir uma solução "do jeito que ela queria" e, bem... você sabe como é, quando você é dev e seu cônjuge é seu stakeholder pessoal, tudo vira um projeto 😄

### A Demanda Original

> *"Eu quero um app para anotar as compras do mercado, mas simples, bonito, e que deixa eu marcar quando já peguei o produto sem descontar do total. Ah, e com rosa. Rosa claro, porque não sou de exagero."*

**Tradução:** Isso virou uma especificação bem clara.

## ✨ Features

- ✅ **Criar múltiplas listas** - Organize suas compras por categoria, mercado ou semana
- ✅ **Adicionar itens rapidamente** - Apenas nome e quantidade (simples assim)
- 💰 **Adicionar valor unitário depois** - Sem pressa, coloca o preço quando souber
- ✔️ **Marcar como "peguei"** - Item fica tachado mas continua no total (porque talvez mude de ideia)
- ✏️ **Editar itens** - Nome e quantidade quando precisar corrigir
- 🗑️ **Excluir itens** - Remove sem dó nem piedade
- 📊 **Totalizadores inteligentes** - Sabe contar quanto você vai gastar
- 🎨 **Design limpo e feminino** - Cores suaves, layout respirado, nada pesado
- 📱 **Totalmente responsivo** - Funciona em celular, tablet e desktop
- 💾 **Dados persistem** - Usa localStorage pra guardar tudo

## 🎯 Como Usar

### Para criar uma nova lista:
1. Clique em "Nova lista"
2. Dê um nome (ex: "Supermercado - Abril/26")
3. Pronto! Já pode adicionar itens

### Para adicionar um item:
1. Coloque o nome do produto
2. Coloque a quantidade
3. Clique em "Adicionar item"
4. Depois coloca o valor quando souber o preço

### Para marcar como "peguei":
- Clique no ✅ ao lado do produto
- O item fica tachado mas continua na conta do total
- *Útil para não se confundir no mercado*

### Para editar:
- Clique no ✏️ e ajuste nome/quantidade

### Para clonar uma lista:
- Na tela principal, clique no ícone de cópia
- Perfeito para compras recorrentes

## 💡 Detalhes Interessantes

### Por que o check não abate do total?

Boa pergunta! No mercado você começa a lista e marca os itens que vai pegando. Mas às vezes você muda de ideia, volta, ou vê algo melhor. O item fica cinzento (tachado) pra não confundir, mas continua no total porque você "ainda" precisa daquilo (ou não, mas a decisão é de quem está nas compras).

### Por que o preço é opcional no início?

Porque nem sempre você sabe quanto custa. Você vai anotando o que precisa, e quando chega no mercado ou vê online, coloca o preço. Simples.

### Por que só uma tela?

Porque o stakeholder pediu simplicidade. Sem menus complicados, sem abas, sem nada que distraia. Entra, vê suas listas, abre uma e compra. Pronto.
